<h1>Introduction</h1>
<p>A cross-platform tomato clock software</p>

<h2>Technology</h2>
<p>vue2 + vuex + vue-router + webpack + electron</p>

<h2>Development</h2>
<pre>
    <code>
    git clone https://github.com/tigergits/tomato-clock.git
    cd tomato-clock
    npm install
    npm run dev
    </code>
</pre>
